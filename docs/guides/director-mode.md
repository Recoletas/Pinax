# 诗歌与散文的编导模式

> 编导模式是本框架面向 **AI 视频创作剧本提示词生成** 的核心能力。

## 1. 模式定位

编导模式将创意写作从纯文本扩展到**视觉叙事**层面：

- **写作模式**：纯文学创作（意象、情绪、叙事）
- **编导模式**：面向 AI 视频生成的剧本提示词（景别、运镜、时长、音效）

## 2. 双模式对比

### 2.1 散文随笔（ProseEssay）

| 功能 | 写作模式 | 编导模式 |
|------|----------|----------|
| 生成 | 意象扩展 | 生成分镜图 |
| 边类型 | 隐喻/并置/断裂/回声 | 跳切/叠化/淡入淡出/对比蒙太奇/交叉剪辑/匹配剪辑 |
| 序列 | 大纲序列 | 时间轴 |
| 节点字段 | 情绪标签 | 景别/运镜/时长/台词/音效 |
| 导出 | Markdown/TXT/JSON | 剪映/Premiere 格式 |

### 2.2 诗歌工坊（PoetryLab）

| 功能 | 写作模式 | 编导模式 |
|------|----------|----------|
| 生成 | 生成灵感树 | 生成分镜图 |
| 边类型 | 隐喻/并置/断裂/回声/层级 | 视觉隐喻/并列镜头/跳切/视觉回声/细节切入 |
| 群组 | 意象群 | 场景组 |
| 节点字段 | 例句/反馈分 | 景别/运镜/时长/色调/声音 |

### 2.3 景别与运镜

| 景别 | 运镜 |
|------|------|
| 远景 / 全景 / 中景 / 近景 / 特写 | 推 / 拉 / 摇 / 移 / 跟 / 固定 |

## 3. 分镜图生成流程

```
用户输入（诗歌/散文）→ AI 分析 → 提取关键意象 →
分配景别/运镜 → 生成节点 → 构建时间轴 → 导出剪映格式
```

## 4. 导出格式

### 4.1 剪映格式（JSON）

```json
{
  "version": "1.0",
  "shots": [
    {
      "sequence": 1,
      "shotType": "extreme_close_up",
      "camera": "fixed",
      "duration": 3,
      "description": "一滴水从叶尖滑落",
      "sound": "水滴声"
    }
  ]
}
```

### 4.2 Premiere XML（EDL）

导出时间线格式，支持直接导入剪辑软件。

## 5. 提示词工程

编导模式的 AI 生成使用分层提示词：

```python
# Layer 1: 场景描述
"描写一个充满诗意的清晨意象"

# Layer 2: 分镜指令
"将上述场景转化为分镜描述，包含景别、运镜、时长"

# Layer 3: 导出格式化
"输出为剪映可用的 JSON 格式"
```
