# 世界书导入与编辑流程

本文档定义“普通入口 + 高级设置”的标准操作链路，适用于产品演示、内容运营与日常开发回归。

## 1. 入口定位

- 普通入口：`/experience/worldbook`
  - 目标：快速起盘，低门槛导入。
  - 能力：预设导入、小说文本导入、AI 根据说明生成草案。
- 高级设置：`/experience/worldbook/advanced`
  - 目标：精细化维护与运营。
  - 能力：条目编辑、注入参数、分组管理、SillyTavern 导入导出。

## 2. 普通入口标准流程

### 方式 A：预设导入
1. 选择世界类型预设。
2. 点击生成预览。
3. 检查条目数量与分组。
4. 导入世界书。

### 方式 B：小说文本导入
1. 粘贴文本，或上传 TXT/MD 文件。
2. 点击生成预览。
3. 系统优先走 AI 提炼；若失败，使用本地提炼兜底。
4. 预览确认后导入。

### 方式 C：说明驱动 AI 生成
1. 输入至少 8 字说明（如世界观、核心冲突、角色关系）。
2. 选择风格与目标条目数。
3. 点击 AI 生成预览。
4. 检查条目命名、关键词和分组后导入。

## 3. 同名冲突策略

当待导入世界书与现有世界书同名时，必须显式选择策略：
- 自动重命名后新建：保留原数据，不覆盖。
- 保持同名直接新建：允许并行同名资产。
- 覆盖同名世界书：替换旧条目，适合“整包重导”。

覆盖前建议确认：
- 现有条目数。
- 导入条目数。
- 条目变化量（正负变化）。

## 4. 高级设置常见动作

- 条目精修：统一维护 `name/type/keys/content/mode/group`。
- 分组运营：创建、重命名、迁移、删除、清理空分组。
- 生态互通：导入与导出 SillyTavern 世界书格式。

## 5. 质量检查建议

导入后建议进行三步自检：
1. 搜索关键词命中率是否正常。
2. 关键角色、地点、事件是否均有条目覆盖。
3. 进入体验页进行至少 1 轮剧情生成冒烟。
