# 文本编辑器与 AI 辅助

> 本模块定位为类 Copilot 的写作辅助工具，核心功能是**导入笔记/小说后由 AI 自动补全或扩展**。

## 1. 核心能力

| 功能 | 说明 | 状态 |
|------|------|------|
| **导入编辑器** | TXT / Markdown 文件导入，保留格式 | 已实现（Writing.vue） |
| **书籍-章节管理** | 新建/删除/切换章节，自动保存 | 已实现（Writing.vue） |
| **富文本编辑** | 排版、分隔线、字体设置 | 已实现（Writing.vue） |
| **AI 补全** | 根据上文续写，类似 Copilot 自动补全 | **规划中** |
| **AI 扩展** | 对选中内容扩展或改写 | **规划中** |
| **导出** | TXT / Markdown 导出 | 已实现 |

## 2. AI 补全设计

### 2.1 交互模式

- **触发方式：** 用户在输入框中输入，AI 实时提供补全建议
- **展示方式：** 灰色显示补全建议，按 Tab 采纳
- **拒绝方式：** 继续输入，灰色建议自动消失

### 2.2 提示词架构

```python
# 系统提示词
{
  "role": "system",
  "content": """
你是一位小说写作助手，擅长根据已有的文本内容提供自然流畅的续写。
续写要求：
- 保持与前文一致的文风、语气、叙事视角
- 不重复已有内容，自然衔接
- 长度适中（50-200字），只提供一段续写
- 用*动作*格式描述动作，"对话"格式描述对话
"""
}

# 用户输入 = 已有文本 + 光标位置标记
{
  "role": "user",
  "content": "已有文本：\n{current_text}\n\n请续写："
}
```

### 2.3 上下文管理

- 只发送当前章节内容 + 最近 N 段历史（约 2000 字上限）
- 重要设定（角色名、地名等）通过构建世界书上下文注入
- 长文本自动截断，保留开头 + 结尾 + 最近段落

## 3. 与世界书系统的联动

编辑器可选择关联世界书：

1. 关联后，AI 续写时自动注入世界书条目（角色设定、地点描述等）
2. 新建角色/地名时可一键存入世界书
3. 编辑器侧边栏显示世界书快速引用

```
编辑器 ──关联──> 世界书
   │                 │
   └── AI 续写时注入设定 ──┘
```
