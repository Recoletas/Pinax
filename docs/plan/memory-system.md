# 05 - 记忆系统

状态：基本完成

## 已完成

- 记忆写入从静默写入改为候选确认流。
- 候选支持 `global-author` / `project` / `session` scope。
- 候选面板支持作用域筛选、计数、编辑后确认。
- 已确认本地记忆会注入小说体验生成上下文。
- mem0 写入适配已接入，未配置时自动跳过。
- mem0 读取兜底已接入，本地无确认记忆时按 scope 搜索远端记忆。
- `useMem0` 支持 scope metadata、scope 检索和缓存隔离。

## 验收状态

| 验收项 | 状态 |
|--------|------|
| 记忆不会自动污染 | 完成 |
| 记忆有明确作用域 | 完成 |
| 不同作品隔离 | 完成 |
| 作者可以看到和控制写入内容 | 完成 |
| mem0 同步/读取兜底 | 完成 |
