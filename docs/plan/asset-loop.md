- 新增统一素材服务 `src/services/narrativeAssets.js`。
- 体验页速记、选段和“整理体验”可以生成素材。
- 写作页素材收件箱已改为详情页式工作面板。

- 世界书草稿可显式写入当前世界书。

- 素材可作为 Copilot 续写参考。
- 素材可进入当前章节纲要，并参与后续内联续写。
