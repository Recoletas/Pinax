# 事故复盘 / Postmortem

> 严重回归的复盘文档。
> **可选**。只在确实有长期价值时建立。
> 它的目的是：让下一个遇到相似症状的人不必再走一遍全链路。

## 格式

每个 postmortem 一个 markdown 文件，命名 `YYYY-MM-DD-<short-slug>.md`：

```markdown
# YYYY-MM-DD: <一句话>

- **影响**: <范围、持续时间、用户可观察后果>
- **根因**: <一两段>
- **时间线**: <发现 → 定位 → 缓解 → 根因修复>
- **教训**: <技术 / 流程>
- **行动项**: <已做 + 待做，链接到 issue / PR>
- **防止复发**: <测试 / 监控 / 文档变更>
```
