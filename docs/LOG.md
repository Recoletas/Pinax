## 2026-05-27 - 诗歌与散文分镜版本卡片

状态：完成首轮

结果摘要：
- 诗歌页分镜侧改成版本卡片，显示镜头数、版本号和校验状态；分镜 Markdown 需要先生成版本，再手动下载。

用户可感知：
- 用户切到分镜时，先看到版本状态而不是一个立即下载动作，源内容和分镜输出的关系更明确。

验证：
- `npm run verify` 通过。

## 2026-05-27 - UI 重构阶段启动

状态：进行中

当前目标：
1. 优先解决用户可用性问题（核心路径、入口数量、操作层级、任务完成）。
2. 再解决兼容阻塞（视口、滚动、层级、浮层冲突）。

已识别风险：
- 页面存在多处固定悬浮入口，交互热区可能重叠。
- 多页面存在 `100vh + overflow + fixed` 叠加策略，移动端易出现遮挡与滚动冲突。
- 页面间断点与布局策略尚未完全统一。
