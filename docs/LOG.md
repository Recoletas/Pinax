## 2026-05-27 - 入口顺序与素材页统一收口

状态：完成首轮

结果摘要：
- 一级导航顺序调整为体验、设定、写作、分镜；体验和写作侧栏的重复切换入口继续收缩。

## 2026-05-27 - UI 视口与固定入口首轮收口

状态：完成首轮

结果摘要：
- 新增全局视口高度变量，工作台壳层和体验、写作、素材、分镜相关页面不再直接依赖固定 `100vh`。

## 2026-05-27 - UI 重构阶段启动

状态：进行中

当前目标：
1. 优先解决用户可用性问题（核心路径、入口数量、操作层级、任务完成）。
2. 再解决兼容阻塞（视口、滚动、层级、浮层冲突）。
3. 在 UI 基线稳定前暂停新增重功能入口。

已识别风险：
- 页面存在多处固定悬浮入口，交互热区可能重叠。
- 多页面存在 `100vh + overflow + fixed` 叠加策略，移动端易出现遮挡与滚动冲突。
- 页面间断点与布局策略尚未完全统一。
