## 1. matching 语义确认

**关键代码**: `src/services/worldbookContextBuilder.js:218`

```js
for (const key of allKeys) {
  const normalizedKey = String(key || '').toLowerCase().trim()
  if (normalizedKey && scanText.includes(normalizedKey) && !matchedKeys.includes(String(key).trim())) {
    matchedKeys.push(String(key).trim())
  }
}
```

**语义**: `scanText.includes(normalizedKey)` — JavaScript 原生 `String.prototype.includes`, **substring includes (case-insensitive)**. 不是 whole-word, 不是 regex.

**含义**:
- keys 必须是玩家 chat text 的 **substring**
- 玩家 "谁阻止我查账" → keys "谁"/"阻止"/"查账" 命中
- 玩家 "追失踪巡骑" → keys "追"/"失踪"/"巡骑" 各自命中, 但 "巡骑失踪" 不命中 (substring 顺序反转)
- 玩家 "天亮前看到下一条真证据" → keys "天亮"/"前"/"看到"/"下一条"/"真"/"证据" 各自命中
