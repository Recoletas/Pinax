# 边境王国 · 雾潮暮湾 内容 review

## 结论

`边境王国 · 雾潮暮湾` 目前足够承担“单旗舰世界”的职责。

它已经具备首轮需要的四个条件：

- 开场困境立即成立，不需要用户先补世界观。
- 势力冲突不是背景板，而是会直接压到玩家选择上。
- 证据链具体，能支撑 10-15 分钟调查型冒险。
- 改写出口清楚，能自然接到“写成作品 / 整理成分镜”。

本轮 review 只看现有种子世界内容、当前 prompt 注入链路和并行计划要求；不改高冲突工程文件。

## 当前事实基线

- 开场困境：`暮湾钟楼连续三夜停摆，北境难民带来“雾中军队”的消息，王室要求你在黎明前确认真相；行会则要求你先封存港口账簿。`
- 内容规模：`6` 个地点、`4` 个势力、`5` 个角色、`3` 个道具、`8` 个事件、`3` 个任务。
- 现有测试已经锁住开局最少注入面：
  - 地点：`暮湾主城`
  - 势力：`潮盐行会`
  - 事件：`钟楼停摆事件`
  - 任务：`黎明前的钟楼调查`
- 改写出口：
  - `写成边境调查章节`
  - `整理成王国势力分镜`
  - `扩展新地点/行会`

## 为什么它能撑住首轮

### 1. 开场不是“介绍世界”，而是“马上选边”

一上来就有三层压力同时成立：

- 时间压力：`黎明前`
- 权力压力：`王室密令`
- 利益压力：`行会账簿`

这比单纯“某地出事了，你去看看”更适合做旗舰入口，因为用户第一轮就知道自己是在做决策，不是在读设定。

### 2. 证据链够具体，GM 不容易空转

这个世界不是抽象谜团，而是已经有一条可以追的调查链：

`钟楼停摆 -> 燃芯失窃 -> 行会夜账 -> 灰墙难民 -> 失踪巡骑 -> 沉钟沼泽`

只要 GM 在前两轮抓住其中任意两段，用户就能感觉到“我在推进一件事”，而不是泛泛聊天。

### 3. 势力差异明确

四个主要势力各自承担不同压力源：

- `潮盐行会`：账、货、港口秩序
- `王家巡骑团`：边境安全、王室执行
- `银藤学院评议会`：异常解释权、观测资料
- `灰墙难民互助会`：第一手证词、道德压力

这让同一个谜题可以从至少三种玩法切入：现场调查、权力谈判、证词追踪。

### 4. 出口和产品主线一致

这个世界天然适合当前执行骨架：

- 玩：调查钟楼、核账、跑灰墙
- 写：边境调查章节
- 分镜：王国势力对峙、港口雾夜、灰墙追踪

它不是只能“聊一段设定”，而是很容易沉淀成后续创作资产。

## 主要风险

### 1. 8 个事件都围绕同一团雾，首屏 3 钩子必须分化

如果入口只给三个差不多的“去调查”按钮，用户会觉得自己只是换了说法，没有换玩法。

这三个首轮钩子必须明确对应不同模式：

- 现场勘验
- 港口博弈
- 难民证词 / 边境追踪

### 2. 前两轮必须给出“代价”或“阻力”

这个世界的氛围很强，但如果 GM 只给雾、钟声、潮湿石墙这些描写，不给明确阻力，调查戏会立刻松掉。

前两轮最低要出现以下之一：

- 谁不想让你看账
- 谁要求你先站队
- 哪条线索会付出代价

### 3. 难民线不能只当背景悲情

`灰墙难民互助会` 和 `苔娜难民领队` 不能只提供“世界更惨了”的氛围，他们必须带出独占信息，否则这条线会被玩家自动判定成次要支线。

## 首屏 3 个行动钩子建议

| 玩法模式 | 按钮文案 | 首轮最低产出 | 不要写成 |
| --- | --- | --- | --- |
| 现场调查 | 先去钟楼查停摆痕迹 | 物证、停摆时间、伊薇队长的调查立场 | 纯背景说明 |
| 港口博弈 | 夜访灯痕码头核对夜账 | 账面矛盾、燃芯流向、行会压力 | 另一种“我去看看” |
| 证词追踪 | 先去难民营找苔娜 | 目击口供、失踪巡骑去向、灰墙线索 | 单纯卖惨的 lore 对话 |

如果后续要统一这三个钩子的文案，应集中在单一入口 helper / adapter 层处理，不要散落在多个页面各写一版。

## 10-15 分钟冒险骨架

建议把首轮体验压成 8 个小拍点：

1. 钟楼停摆，给出王室与行会的双重命令。
2. 玩家选择第一条线：钟楼 / 账簿 / 难民。
3. 第一现场出现能验证的异常，而不是只给传闻。
4. 第二个势力进场施压，逼玩家不能两边都要。
5. 玩家拿到第一手证据或证词。
6. 证据把故事从主城推到第二地点：码头、仓库或灰墙。
7. 玩家遇到一次明确的取舍：封锁、保护证人、继续追踪。
8. 收尾时给出下一段悬念，最好指向 `沉钟沼泽` 或 `灰墙真相分岔`。

做到这 8 步，基本就能保证 10-15 分钟不是空转。

## 内容线程建议

如果内容 review 后续需要落 patch，优先顺序应是：

1. 只改种子世界内容和入口文案。
2. 先补“首轮差异化行动钩子”。
3. 只有确认内容不够支撑时，才提工程层改动需求。

先不要直接去碰：

- `src/pages/Experience.vue`
- `src/pages/WorldBookQuickImport.vue`
- `src/stores/gameStore.js`

## 最终判断

`边境王国 · 雾潮暮湾` 不是“设定最多”的世界，但它是当前最像一个能直接玩的入口世界。

它的重点不是世界观广度，而是：

- 第一轮就有压力
- 第二轮就有证据
- 第三轮就要选边

这正好适合当前“选择世界 -> 开始冒险 -> 写成作品 -> 整理成分镜”的主线。
