# 边境王国 · 雾潮暮湾 demo case

## 用途

这份文档给三个用途共用：

- 并行内容线程的手测执行稿
- 工程线程做入口 / 体验页时的样例目标
- 后续营销或演示时的最小故事闭环

目标不是证明“世界很多”，而是证明一件事：

`选世界 -> 玩 10-15 分钟 -> 拿到可写素材 -> 能改成章节 / 分镜`

## 一句话 demo

暮湾钟楼停摆的第三夜，一名调查者必须在黎明前查清真相；但王室要他先封锁难民，行会要他先封存账簿，而真正的证据正沿着雾港、灰墙和沉钟沼泽向外扩散。

## 手测口径

本轮文档给的是一份 `12 分钟执行稿 + 示范记录稿`。

执行者按当前链路去跑时，只要满足以下标准，就算通过：

- 第一轮或第二轮明确引用世界书中的地点 / 势力 / 事件 / 任务
- 不需要用户自己补设定才能继续玩
- 过程中至少产出 3 条可沉淀为写作素材的句子或节点
- 结尾能自然接到“写成我的版本”或“整理成分镜”

如果真实 GM 回复和下面样稿不完全一致，只要满足上述口径，仍算通过。

## 示范记录稿

下面不是唯一正确输出，而是一份可供对照的样稿。

### Round 1

玩家输入：

> 我先去钟楼查停摆痕迹。

GM 最低应做到：

- 明确地点是 `暮湾主城` 的钟楼区域
- 把 `钟楼停摆事件` 说成正在发生的危机，不是历史背景
- 让 `伊薇队长` 或同级别现场权力角色出场

可沉淀素材：

- “钟楼沉默得像被潮水含住了舌头。”

### Round 2

玩家输入：

> 我要看最近三夜的值守记录，尤其是每次停摆前后谁离开过钟楼。

GM 最低应做到：

- 给出一条可验证异常，例如值守记录缺页、时间戳冲突或失踪名单对应停摆时刻
- 让现场调查变成证据链，不只是氛围描写

可沉淀素材：

- “每一次停摆，都对应一笔被划掉的名字。”

### Round 3

玩家输入：

> 接着去灯痕码头核对行会夜账。

GM 最低应做到：

- 引用 `潮盐行会`
- 让 `雾灯燃料失窃` 和账簿异常连上
- 出现“有人不想让你继续查”的阻力

可沉淀素材：

- “夜账写着燃芯仍在仓，可码头上的铁锁却像刚被海盐磨开。”

### Round 4

玩家输入：

> 谁阻止我查账，我就先记下谁的名字，再看他怕我看到什么。

GM 最低应做到：

- 给出一个具体的行会阻力人物或代理人
- 明确“封账”不是建议，而是有后果的命令

可沉淀素材：

- “账房里没有风，纸页却像有人在下面轻轻翻它。”

### Round 5

玩家输入：

> 我去难民营找苔娜，确认雾中军队的目击。

GM 最低应做到：

- 引用 `苔娜难民领队`
- 让 `灰墙难民互助会` 带出独占证词
- 把 `灰墙巡骑失踪` 作为新线索推出

可沉淀素材：

- “苔娜说那支军队没有旗，只有雾替他们走在前面。”

### Round 6

玩家输入：

> 把难民证词和夜账流向对一下，看它们是不是都指向灰墙。

GM 最低应做到：

- 让两条线索交汇
- 给出下一地点：`北境灰墙`、`雾灯仓库` 或 `沉钟沼泽`
- 把故事从单点调查推进到路线决策

可沉淀素材：

- “一边是被海盐染白的账页，一边是被雾水泡软的口供，它们都指向同一面旧墙。”

### Round 7

玩家输入：

> 如果王室密令现在要我封锁难民营，我先保住证人，再追失踪巡骑。

GM 最低应做到：

- 明确 `王室密令抵达`
- 明确这是一次立场选择，不是免费兼得
- 让势力关系发生变化

可沉淀素材：

- “黎明还没来，命令却先来了，而且它比钟声更冷。”

### Round 8

玩家输入：

> 带我去还能追上巡骑痕迹的地方，我要在天亮前看到下一条真证据。

GM 最低应做到：

- 把结尾推向 `灰墙真相分岔` 或 `沉钟沼泽`
- 给出明确的下一段悬念
- 让玩家感到这段冒险可以继续，也可以在这里沉淀为作品

可沉淀素材：

- “远处的沼泽像一口倒扣的钟，等着把最后一声回响还给暮湾。”

## 冒险摘要样例

你在暮湾钟楼停摆的第三夜接手调查，先在主城钟楼发现停摆时刻与失踪记录一一对应，又在灯痕码头查到雾灯燃芯的夜账异常。随后你从苔娜处确认，灰墙外确实出现过没有军旗、只被雾潮簇拥的队列。王室密令要求立即封锁难民营，行会则要求你停止核账；你最终选择保住证人并追查失踪巡骑，把线索一路推向灰墙与沉钟沼泽。

## 小说化样例

钟楼停摆后的暮湾比平时更安静，像整座城都在等谁先开口。伊薇队长把值守册摊在潮湿的石台上，指尖压着那几处被反复刮擦的时刻。每一处空白，都对应一个再也没有回来的名字。等我离开钟楼去往灯痕码头时，雾已经把港口的灯洗成一层发灰的光，潮盐行会的账房却亮得刺眼，仿佛里面的人不是在守账，而是在守一件不该被看见的事。等苔娜把那句“他们没有旗，只有雾替他们走在前面”低声说出来，我忽然明白，这一夜真正被封锁的从来不是难民营，而是有人拼命想压住的证据链。

## 分镜节点清单

1. `暮湾钟楼外景`
   - 远景，夜雾压城，钟楼无声
   - 作用：建立异常和时间压力
2. `伊薇翻开值守册`
   - 中景，潮湿纸页、指尖按住被刮掉的时刻
   - 作用：第一条可验证证据
3. `灯痕码头夜账房`
   - 室内对峙，账册、盐灯、紧闭铁锁
   - 作用：把谜题转成利益冲突
4. `行会代理人阻拦查账`
   - 近景，低声施压，命令先于事实到场
   - 作用：让玩家感到代价
5. `难民营里的苔娜`
   - 半身景，风吹湿篷布，证词压低声音
   - 作用：给出独占信息
6. `账页与口供并置`
   - 插入镜头，两份证据在桌上并排展开
   - 作用：完成线索交汇
7. `王室密令抵达`
   - 特写，封蜡、盐痕、命令内容
   - 作用：制造取舍
8. `灰墙或沉钟沼泽的天亮前追踪`
   - 远景转跟拍，雾中只剩痕迹和钟声回响
   - 作用：把这一段收成下一幕钩子

## 这个 demo 应该传达什么

不是“我们有很多世界”。

而是：

- 你进入一个已经能玩的世界
- 第一段冒险能自己长出证据、冲突和素材
- 结束后不是只能继续聊天，而是已经能写成章节或拆成分镜

### 开场钩子需要补什么

1. 第一条钩子现在方向对，但标题不够尖。
   - 当前 `buildPlayableWorldActionHooks()` 里第一条标题会落到 `暮湾主城`，这对入口卡面太宽。
   - 这里更适合直接落成 `暮湾钟楼` 或 `钟楼值守室`，让玩家一眼知道第一现场。

2. 第一条钩子的 detail 还不够像“开局动作”。
   - 现在是“把开场困境落到可验证的证据”。
   - 更需要把 `停摆时刻 / 值守记录 / 失踪名单` 这些具体证据词抬出来，这样 GM 第一轮更容易直接命中证据链。

3. 第二条钩子结构可用，但还差一个明确阻力人。
   - `夜访码头核夜账` 已经能把 `潮盐行会` 和账本异常拉上桌。
   - 但世界书里还缺一个“行会账房代理人 / 码头夜班头目”之类的人物，GM 很容易只给组织压力，不给可对峙对象。

4. 第三条钩子是三条里最稳的，但标题重心放错了。
   - 当前 title 会偏向 `苔娜难民领队`。
   - 开场卡面更适合把场景放前面，例如 `灰墙难民营`，再把苔娜放进 detail 或 command。

### 世界内容需要补什么

1. 缺一个显式的“钟楼”微地点条目。
   - 现在有 `暮湾主城`，但没有 `暮湾钟楼` / `钟楼值守室`。
   - 这会让第一轮容易掉回“大主城氛围描写”，而不是直接落到钟楼现场。

2. 缺一个显式的“难民营”微地点条目。
   - 现在 `北境灰墙` 很像区域，不像可立即进入的现场。
   - 如果补 `灰墙难民营` 或 `灰墙临时营地`，第五轮“找苔娜”会更稳。

3. 钟楼线和学院线之间还缺一个可被 GM 反复引用的硬证据。
   - 现有 `学院观测仪过载`、`钟楼停摆事件`、`沉钟残片` 各自都成立。
   - 但三者之间最好再补一个桥，例如“被刮掉的观测曲线编号正对应钟楼停摆时刻”这类能反复引用的关联句。

4. 行会线缺一个可立刻施压的角色。
   - `塔维克书记官` 更像旧账知情人，不像当场阻拦的人。
   - 建议补一个会直接挡查账、封账、拖时间的人物，给第四轮明确反作用力。

5. 结尾分岔已有方向，但即时损失还不够具体。
   - `灰墙真相分岔` 已经说明会改势力关系。
   - 还应再明确 1-2 个“当场要失去什么”，比如失去证人、失去账本窗口、失去巡骑追踪时机，这样第七轮更容易形成真正抉择。

### 写回出口文案需要补什么

1. `写成边境调查章节` 可以保留。
   - 这是当前三条里最直接、最像用户产物的一条。

2. `整理成王国势力分镜` 方向对，但还不够像可立即产出的结果。
   - 更适合改成 `拆成 8 格边境悬疑分镜` 或 `整理成三方对峙分镜`。

3. `扩展新地点/行会` 太偏内容编辑，不像玩家完成一段冒险后的出口。
   - 这条更像作者后台动作，不像体验结束后的写回按钮。
   - 更适合改成 `整理成王室 / 行会 / 难民三方证据卡`、`写成灰墙支线大纲` 之类能直接接写作的出口。

### 拿到可用模型后要重点验证什么

1. Round 1 是否真的会先落 `钟楼`，而不是泛泛描述 `暮湾主城`。
