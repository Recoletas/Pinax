# 功能与工作流

九大功能逐个讲，然后四个典型工作流串起来走一遍。

## 1. 欢迎页（`/`）

第一次打开看到的就是这个。卡片式入口分到五个工作区：体验 / 世界书 / 写作 / 素材 / 散文画布。右上角齿轮进 **设置**。

主要动作：

- 点工作区卡片进入对应功能
- 齿轮进设置

## 2. 体验（`/experience`）

整个项目的核心玩法页。聊天面板 + 状态栏 + 机制面板 + 里程碑弹窗。

主要动作：

- **选世界** —— 第一次进入会让你从五个预设里选一个（仙侠 / 奇幻 / 末日 / 科幻 / 都市）
- **输入行动** —— 文本框回车，AI 接着写
- **机制按钮** —— 右下角 combat / trade / quest / dialogue
- **里程碑** —— 弹窗时点"接受"或"忽略"
- **换世界** —— 顶部栏能切回世界书或选其他世界

## 3. 世界书（`/experience/worldbook` + `/experience/worldbook/advanced`）

两入口设计：

- **普通入口** `/experience/worldbook` —— 快速起盘，三种导入方式（预设 / 文本 / AI 说明）
- **高级设置** `/experience/worldbook/advanced` —— 精细化维护：条目编辑、分组、SillyTavern 互通用

主要动作：

- 导入：选预设 / 粘文本 / 写一段说明让 AI 生成
- 编辑：增删改条目、分组、关键词
- 冲突处理：同名世界书有三种策略（重命名 / 同名并存 / 覆盖）
- 互通用：SillyTavern 格式导入导出

> 详细流程：[世界书工作流指南](../guides/worldbook-workflow.md)

## 4. 结构化设定（`/experience/settings/structured`）

字段化的朝代 / 角色 / 地点 / 场景。能画表、改下拉菜单。详见 [02-concepts.md](./02-concepts.md) 的"结构化设定"。

## 5. 世界地图（`/experience/world-map`）

基于 Voronoi 的程序生成地图，能看到地形、气候、生物群系、文化圈、势力范围、道路网络。地图数据很大时（比如 Pangea 种子的整块大陆）生成会比较慢，这是正常的。

调试用：按 `?` 或 DevTools 控制台输入 `window.__PERF__` 可以看各阶段耗时。

## 6. 写作（`/writing`）

小说写作主界面。左侧书 / 章列表，中间编辑器，右侧 AI 工具栏。

主要动作：

- 新建书 / 新建章
- 在 `InputArea` 写正文
- 触发 **文本扩展**（从一段扩成一段）/ **改写**（润色、换风格）/ **续写**（AI 接着往下）
- 选段导出 **章节分镜**
- 自动保存到 `localStorage`

## 7. 素材（`/materials`）

写作过程中的金句 / 重要事件 / 关键描写存到这里。每条带情绪标签。素材池也是顾问和散文画布的输入。

## 8. 散文画布（`/prose-essay`）

节点-边-时间轴三件套的画布。素材 → 卡片 → 时间轴 → 分镜版本，整条流水线在这页能完成。

主要动作：

- 创建节点（卡片）
- 连边（前置 / 后继 / 引用 / 镜像）
- 排时间轴
- commit / branch（分支）/ pile（堆叠）
- 导出分镜

## 9. 图片生成（侧栏 `ImageGenRail`）

不是单独页面，是右侧栏的 AI 生图。绑多个 provider 配置，键名 `image_model_configs`。

## 10. 顾问（章节旁的顾问按钮）

不是单独页面，是写作时调用 OpenClaw 网关的体检功能。返回 critical / warning / info 三档提示。要用得先在 [04-configuration.md](./04-configuration.md) 配 OpenClaw。

## 11. 设置（齿轮图标）

三标签：

- **通用** —— 语言（中文 / 英文）、文字速度（快 / 正常 / 慢）、自动存档开关
- **AI API** —— provider / apiKey / baseUrl / model / 测试连接
- **记忆系统** —— Mem0 apiKey / host / 测试连接

# 典型工作流

## 工作流 1：第一次跑通

[01-quickstart.md](./01-quickstart.md) 走的就是这条：

1. 欢迎页 → 设置 → 选 provider → 填 apiKey → 测试 → 保存
2. 切到"记忆系统"标签 → 填 Mem0 apiKey（可选，没填也能用）→ 保存
3. 欢迎页 → 开始体验 → 选预设世界 → AI 出开场白 → 输入行动 → 接着玩

## 工作流 2：搭一本自己的世界书

适用于你要写一个"非预设"的世界：

1. 欢迎页 → **世界书**（普通入口）
2. 选"方式 C：说明驱动 AI 生成"
3. 写一段至少 8 字的说明，比如：
   > 修真 + 蒸汽朋克混合。1890 年代东方大陆，修士用灵石驱动机甲。
4. 选风格、目标条目数
5. AI 生成预览 → 检查条目命名、关键词、分组
6. 导入世界书
7. （可选）切到 **高级设置** 精修：调关键词、改条目正文
8. 进 **结构化设定** 建朝代、角色、地点
9. 回 **体验** 选这个新世界，开写

## 工作流 3：写一章带分镜的小说

1. 欢迎页 → **小说写作**
2. 新建书（比如《蜀山剑侠》）→ 新建章（"第一章·出山"）
3. 在 InputArea 写开头，比如：
   > 李青云站在山门前，回望了一眼雾气中的主峰。
4. 选中刚写的那段 → 点 **续写** → AI 接着往下铺
5. 不满意就再点 **改写**，换一种风格
6. 整章写完后 → 选全文 → **导出分镜**
7. 切到 **散文画布**，分镜自动落成节点
8. 排时间轴，调节点顺序、连边
9. （可选）点 **顾问** 让 OpenClaw 做一次体检

## 工作流 4：把世界书偏好"教"给 AI

这个走 Mem0 路线，跨会话都生效：

1. 在 **设置 → 记忆系统** 填好 Mem0 apiKey
2. 正常写作
3. AI 写作时如果识别到"作者偏好"类信息（"我喜欢短句" / "不要写战斗细节"），会推到 **记忆候选池**
4. 候选池里出现弹窗时，点 **采纳**
5. 采纳后这条进 Mem0。下次开新会话，AI 读 prompt 时会带上这条偏好

要查看已有的 Mem0 记忆：[mem0.ai](https://mem0.ai) 后台 → Memories。
